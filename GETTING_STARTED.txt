⚡ QUICK START
───────────────────────────────────────────────────────────────
1. Open Terminal and navigate to project:
   cd /Users/apple/business_card_app

2. Install dependencies:
   flutter pub get

3. Run the app:
   flutter run

4. Build APK:
   flutter build apk --release

📱 FIRST TIME USAGE
───────────────────────────────────────────────────────────────
1. Create an account (Sign Up)
2. Sign in with your credentials
3. Tap "Add Card" button
4. Choose Camera or Gallery
5. Review mock OCR data
6. Save the card
7. View, search, edit, or delete cards

🆘 TROUBLESHOOTING
───────────────────────────────────────────────────────────────

Issue: Camera not working
Solution: Check permissions in AndroidManifest.xml/Info.plist

Issue: Hive errors
Solution: flutter pub run build_runner build --delete-conflicting-outputs
